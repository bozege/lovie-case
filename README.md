# Lovie Case

Peer-to-peer payment request app built for the Lovie interview assignment.

## Setup

1. Copy `.env.example` to `.env`
2. Install dependencies with `npm install`
3. Start the app with `npm run dev`

## Available Scripts

- `npm run dev`
- `npm run build`
- `npm run test`
- `npm run test:e2e`

## Environment Variables

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
